Features of every site

Each html file has its own feature and functions:

  1. HomePage: It can take you to all the other pages. It has links both on the navigation bar in pink and in the breadcrumbs section.
  2. Sign Up: Sends the data to a server to be processed in this case my email. The terms and conditions can be viewed here. It also has breadcrumbs on top.
  3. Sign In: Sends the data to a server to be processed in this case my email. It also has breadcrumbs on top.
  4. Design Summary:
  5. About Me: Contains information about me as the designer as well as links to pages of my gallery where I stored the images and videos that I liked and my portfolio where I have my school work stored in. It also has breadcrumbs on top.
  6. Guest Book: Used to receive feedback from customers after signing up.It also has breadcrumbs on top.
  7. Samsung/iPhone/Huawei/Oppo/Infinix: Contains the list of the respective choice of phone. Clicking the images or words next to the image should take you to the next page where you can view more details about the phone. It also has breadcrumbs on top.
  8. Specific Product Details: Contains the details of the respective phone that has been selected. The user can now select the number of such products that he wants and click on BUY NOW to start the process of buying it which will lead to a form called Shipping Address. It also has breadcrumbs on top.
  9. Specific Shipment Form: Contains a form which needs to be filled by the user concerning his address, after which, he can then click on the Next button to continue to step 2. It also has breadcrumbs on top.
  10. Specific Order Summary & Payment page: The user confirms that the product shown is what he selected and pays through M-PESA using the details provided on the page. After he is done, he can click on the Done Button which will take him to a page showing that he's product is on the way. It also has breadcrumbs on top.
  11. Complete Purchase Page: This page shows where the product is from the store to reaching the customer to giving comments. It also gives a link to give feedback by the user through the Guest Book page and another link to for the user to view other products. It also has breadcrumbs on top.

CSS features used

I have used mostly Internal CSS and a little bit of In-Line CSS.

  • For the body, I used:
      margin: 0; so that all there's is no default margins set by the browser.
      font-size: 100%; so that all font go in accordance with the person's display.
      background-color: #eee000; to change the background-color to yellow.

  • For the h1 tag (the first heading) I used:
      color: #ee0000; so that I can capture the eye in red color.
      background-color: #dddddd; so that I change the background of the first heading to grey.
      margin: 0; to remove any default margins in the browser on the first heading.
      border-bottom: 0.3em solid #888888; to create a dark grey border at the bottom of the first heading.
      border-radius: 0em 0em 1em 1em; to make the border at the bottom rounded.
      text-align: center; to make the first heading to align at the center.
      padding-left: 18%; to move the first heading to the center of the page by accounting for of the logo.
      padding: 1em; to create a bigger space for the content of the first heading.
      font-family: "Apple Chancery", cursive; to change its font to a cursive one called "Apple Chancery."

  • For the images, I used:
      height: 70px; to change its height.
      float: left; to make it float on the left.
      margin-left: 7%; to add a margin on the left side.
      margin-top: 1em; to add a margin on the top side.

  • For the third headings, I used:
    margin-left : 5%; to add a neat space on the left so that the content doesn't look tight.

  • For all the links in general, I used:
      text-decoration: none; to remove the uunderlining of links.
      a:link { color: #007700; } : to change the color of the links to green.
      a:focus { color:#8985a0; } : to change the color to slightly yellowish when it's highlighted/focused.
      a:hover { color: #ff0000; } : to change the color to red when a cursor moves infront of it.

  • For the breadcrumbs, I used:
      margin-bottom: -3em; to reduce the wasted space below it.
      margin-top: 1em; to increase the space between it and the content of the first heading for neatness.

  • For the search text box, I used:
      width: 25%; so that its size is relative to the device of the user.
      padding: 8px; so that it doesn't look squeezed up but to allow the space between it and other contents.
      padding-top: 0px; so that it stays on the same line as the breadcrumbs.
      float: right; so that other content can stay on its left and that it rises to its position.
      margin-right: 10%; so that it doesn't touch the right margin however big or small the screen of the user might be.
      margin-top: -1.5em; so that it is on the same line as the breadcrumbs.

  • For the submit button near the search text box, I used:
      background-color: #eee000; to make the background the same color a the background of the whole html file.
      color: #ee0000; so that I can change the text color to red.
      border-color: #007700; so that the border is visible and in a different colour, in this case, color green.

  • For the information stored at the center of the page, I used:
      color: #000000; to change the text color to black.
      background-color:#dddddd; to create uniformity by making the background the same colour as the background of h1.
      text-align: left; to align the text on the left side.
      height: 364px; to set and change the height of the content on the center so taht I can control its background.
      width: 800px; to set and change the width of the content on the center so taht I can control its background.
      float: center; to make everything be floating at the center so that content can be in both left and right.
      margin-top: -21em; to reduce the wasted space on the top if this is not applied.
      margin-left: 17.1em; to move the content in place at the center.
      margin-bottom: 1em; to add a margin between this content and the contach information below it.
      border: 3px outset #dddddd; to make the content seem to get outof the webpage and set the border to grey.
      padding: 2em; to add more space in the content between its border to make it neat.
      padding-top: 0em; to remove the wasted space on top.

  • For improving the breadcrumbs further, I also added the following to the li tag:
      float: left; to make them move to the left side of the page.
      display: block; to make them acts as individual content within the whole section of breadcrumbs.
      background-color: #dddddd; to make the background of the breadcrumb grey.
      border: 3px ridge #888888; to make the border easily visible.
      padding: 5px; to make the contents inside the breadcrumbs to be neat and not near tight near the border.
      margin: 3px; to add some space between the breadcrumbs to distinguish each breadcrumb from each other.
      border-radius: 20px; to make the borders have round edges.

  • For every image at the center, I used:
      height: 175px; to make their height smaller and the same for uniformity.
      width: 175px; to make their width smaller and the same too for uniformity.
      margin-right: 2em; to make space between the images and other content and images.
      margin-top: -0.5em; to reduce the wasted space on top of the images.
      padding: 0em 1.5em 1.5em 1.5em; to make the images have ample space between them.

  • For the product navigation on the left, I used:
      margin: 5%; to make it not touvch the left margina and for neatness.
      height: 250px; to make the height fixed so as to apply other styles such as background-color.
      width: 150px; to make the width fixed so as to apply other styles such as background-color.
      color: #007700; to change the color of the text.
